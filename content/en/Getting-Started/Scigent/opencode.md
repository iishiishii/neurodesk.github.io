start open code on a terminal with `opencode`

By default OpenCode is setup to use gpt-oss-120b. This model works when running play-america.neurodesk.org or edu.neurodesk.org. When running on a different instance configure your model provider:
- ctrl+p 
- and select switch model 
- then ctrl+a to connect a different provider
- openai works well for example
